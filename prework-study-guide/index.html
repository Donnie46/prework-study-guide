Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • All the elements that we want the user to see and interact with must be inside the body.
  • Each element is marked with an opening bracket and a closing bracket. The closing is accompanied with "/".

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element. When you increase padding, you are making the element bigger.
  • There are 3 ways to style a webpage using CSS: Inline CSS, Internal CSS Style Sheet, and the most common way is External CSS style sheet.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • A declaration contains 2 important components, the CSS property we want to apply and the value of the property.
  • If we want to apply a rule to all elements on the page, we insert a "*" symbol.
  • CSS determines the styles of a webpage in order to make a website visually appealing. CSS can also make a website simpler to use with a polished design and consistent organization.

Git

  • git status: checks what branch we are currently in
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: lets us swith to the main branch
  • git add -A: lets us stage the changes so we can add the changed files in our working directory of Git
  • touch lets us create a file in a directory
  • cd: command to change directories
  • pwd: shows us what directory we are currently in
  • ls: lists all files inside the directory we are currently in
  • The "ul", "ol", and "li" elements represent unordered lists, ordered lists, and list items. (Bulleted or numbered lists).
